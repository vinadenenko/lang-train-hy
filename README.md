# Armenian Learner

A React app for learning the Eastern Armenian alphabet and vocabulary.

## Requirements

- Node.js 18+
- npm 9+

---

## Install dependencies

**Windows (PowerShell)**
```powershell
npm install
```

**Windows (cmd)**
```cmd
npm install
```

**Linux**
```bash
npm install
```

---

## Run dev server

**Windows (PowerShell)**
```powershell
npm run dev
```

**Windows (cmd)**
```cmd
npm run dev
```

**Linux**
```bash
npm run dev
```

Then open http://localhost:5173 in your browser.

---

## Build for production

**Windows (PowerShell)**
```powershell
npm run build
```

**Windows (cmd)**
```cmd
npm run build
```

**Linux**
```bash
npm run build
```

Output goes to the `dist/` folder.
